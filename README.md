# Генератор резюме

Простое Flask-приложение для генерации текста резюме на основе введённых пользователем данных.

## Возможности

- Ввод имени, вакансии, компетенций, возраста и стиля изложения
- Три стиля: серьёзный, остроумный, безграмотный (с возрастными вариациями)
- Валидация полей — пустые или бессмысленные данные не принимаются
- Автоматический контроль длины текста (1500–5000 значимых символов)
- Современный адаптивный интерфейс

## Запуск

```bash
pip install flask
python app.py
```

Открой `http://127.0.0.1:5000` в браузере.

## Технологии

Python, Flask, HTML, CSS, JavaScript
